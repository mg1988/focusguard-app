## 问题一：Guideline 2.3.6 - 年龄分级设置不准确

### 问题描述
在App Store Connect中，年龄分级设置了"家长控制"和"年龄确认"为"有"，但实际应用中并没有这些功能。

### 整改步骤

1. 登录 [App Store Connect](https://appstoreconnect.apple.com)
2. 选择你的应用 "FocusGuard"
3. 进入"App信息"页面
4. 找到"年龄分级"部分
5. 点击"编辑"
6. 将"家长控制"和"年龄确认"都改为"无"
7. 保存更改

### 验证
- 确认年龄分级显示为4+
- 确认"家长控制"和"年龄确认"都显示为"无"

## 重新构建和提交

### 步骤1：重新构建应用

```bash
# 1. 打开Xcode项目
open focus_mac.xcodeproj

# 2. 在Xcode中：
#    - 选择 Product → Scheme → Edit Scheme
#    - 配置：Release, Any Mac (Apple Silicon, Intel)
#    - Product → Clean Build Folder (⇧⌘K)
#    - Product → Archive

# 3. Archive完成后：
#    - 点击 "Validate" 验证构建
#    - 点击 "Upload to App Store Connect" 上传
```

### 步骤2：更新App Store Connect信息

1. 登录 [App Store Connect](https://appstoreconnect.apple.com)
2. 选择应用 "FocusGuard"
3. 进入"App信息"页面
4. 修改以下字段：
   - **名称**: `FocusGuard`
   - **副标题**: `AI 驱动的效率提升工具`
   - **应用描述**: 使用上面的中文描述
   - **英文描述**: 使用上面的英文描述
   - **版本发布说明**: 使用上面的版本说明
5. 进入"年龄分级"部分
6. 将"家长控制"和"年龄确认"改为"无"
7. 保存所有更改

### 步骤3：提交审核

1. 在App Store Connect中，进入"版本"页面
2. 确认所有信息已更新
3. 点击"提交以供审核"
4. 填写审核说明（见下方）

## 注意事项

1. **不要修改 Bundle Identifier**
   - 保持 `com.mg.focus-mac` 不变
   - 修改 Bundle ID 会导致用户无法升级，相当于发布新应用

2. **保留所有功能代码**
   - 本次整改只修改元数据和名称
   - 不要修改应用功能代码

3. **截图可以保留**
   - 当前截图没有问题，可以继续使用

4. **隐私政策和使用条款**
