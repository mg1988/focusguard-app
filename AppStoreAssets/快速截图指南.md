## ✅ 第一步：运行应用（1 分钟）

### 在 Xcode 中：

```bash
# 打开项目
open focus_mac.xcodeproj
```

然后：
1. 选择目标设备：**My Mac**
2. 点击 **Run** 按钮（或按 `⌘R`）
3. 等待应用启动

应用启动后，会在**屏幕顶部菜单栏**显示一个小图标。

## 📐 第三步：调整尺寸（1 分钟）

### 使用 Preview 应用（最简单）

1. **双击打开截图**

2. **调整尺寸**
   ```
   菜单：Tools → Adjust Size
   ```

3. **输入尺寸**（单位：像素）

   **MacBook Pro 13"**：
   - 宽度：`2880`
   - 高度：`1800`
   - ✅ 勾选 "Scale proportionally"

   **iPad Pro 12.9"**：
   - 宽度：`2048`
   - 高度：`2732`

4. **保存**
   ```
   File → Save (S)
   ```

##  常见问题

### Q: 截图有菜单栏怎么办？
A: 使用 ` + ⇧ + 4 + 空格` 只截取窗口，不要截全屏

### Q: 统计数据是空的怎么办？
A: 先使用应用 2-3 天积累数据，或联系开发添加测试数据

### Q: 截图尺寸总是不对？
A: 确保在 Preview 的 Adjust Size 中勾选了"Scale proportionally"

### Q: 如何确认截图尺寸？
A: 右键截图 → 显示简介 → 查看尺寸信息
