**步骤**：
1. 创建 GitHub 仓库：`focusguard-web`
2. 上传 `privacy.html` 和 `terms.html`
3. 启用 GitHub Pages
4. 获得网址：`https://yourname.github.io/focusguard-web/privacy`

**优点**：
- ✅ 完全免费
- ✅ 专业域名
- ✅ 稳定可靠

### 方案 2：Notion 公开页面（最简单）

**步骤**：
1. 在 Notion 创建隐私政策页面
2. 点击右上角 "Share"
3. 开启 "Share to web"
4. 复制公开链接

**优点**：
- ✅ 3 分钟完成
- ✅ 无需技术
- ✅ 可随时更新

### 方案 3：飞书文档（国内最快）

**步骤**：
1. 在飞书创建文档
2. 粘贴隐私政策内容
3. 点击"分享"
4. 开启"互联网可访问"
5. 复制链接

**优点**：
- ✅ 国内访问快
- ✅ 支持中文
- ✅ 即时生效
