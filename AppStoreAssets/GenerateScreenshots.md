## 🛠️ 截图工具推荐

### 方案 1: Xcode 内置截图（推荐）

**步骤**:
1. 在 Xcode 中运行应用
2. 点击菜单栏 **File → New Simulator Window**
3. 选择 **Mac** 设备
4. 运行应用到模拟器
5. 在模拟器中点击 **File → Capture Screen** (⌘S)
6. 截图会自动保存到桌面

**优点**:
- 截图尺寸标准
- 自动保存为 PNG
- 无菜单栏干扰

### 方案 2: CleanShot X（专业工具）

**下载**: https://cleanshot.com

**优点**:
- 支持设备边框
- 自动标注尺寸
- 批量处理
- 专业美观

### 方案 3: macOS 原生截图

**快捷键**:
- `⌘ + ⇧ + 3`: 截取全屏
- `⌘ + ⇧ + 4`: 截取选定区域
- `⌘ + ⇧ + 4 + 空格`: 截取窗口

**获取窗口截图**:
1. 按 `⌘ + ⇧ + 4 + 空格`
2. 鼠标移到应用窗口（会高亮）
3. 点击窗口
4. 截图保存到桌面

## 📐 截图尺寸调整

如果截图尺寸不符合要求，可以使用以下方法调整：

### 使用 Preview（预览）

1. 打开截图
2. `Tools → Adjust Size`
3. 输入目标尺寸（如 2880 x 1800）
4. 确保 "Scale proportionally" 已勾选
5. 保存

### 使用命令行

```bash
# 调整尺寸为 MacBook Pro 13" (2880x1800)
sips -z 1800 2880 screenshot.png --out screenshot_resized.png

# 调整尺寸为 iPad Pro 12.9" (2048x2732)
sips -z 2732 2048 screenshot.png --out screenshot_resized.png
```
