## 🎯 提交步骤

### 步骤 1：准备构建

```bash
# 1. 打开 Xcode
open focus_mac.xcodeproj

# 2. 选择 Product → Scheme → Edit Scheme
# 3. 配置：Release, Any Desktop (Apple Silicon, Intel)

# 4. Product → Archive
# 5. 验证构建
# 6. 上传到 App Store Connect
```

### 步骤 2：填写 App Store Connect

1. 登录 https://appstoreconnect.apple.com
2. 选择"我的 App" → "+" → "新 App"
3. 填写所有信息（见上文）
4. 上传应用图标和截图
5. 提交审核

### 步骤 3：等待审核

- 审核时间：3-7 个工作日
- 审核状态可在 App Store Connect 查看
- 如有驳回，根据反馈修改后重新提交
