## 常见问题与解决方案

### 1. 无效字符列表

App Store 不允许的字符：
- 某些特殊符号（如版权符号©、注册符号®使用不当）
- 超链接（如 http://, https://）
- HTML 标签
- 控制字符
- 某些 Unicode 特殊字符

### 2. 正确做法

- 使用纯文本描述
- 不要在描述中添加链接（URL 放在专门字段）
- 避免使用 emoji 或特殊符号
- 不要使用™或®符号
- 标题中不要使用特殊字符

### 3. 描述格式建议

每段建议：
- 段落之间用空行分隔
- 列表项用短横线"- "开头
- 不要使用编号列表
- 保持简洁，避免过长描述
