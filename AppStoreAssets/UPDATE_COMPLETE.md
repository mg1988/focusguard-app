## ️ 常见问题

### Q1: 提示 "fatal: remote origin already exists"

**解决**：
```bash
# 删除远程仓库重新添加
git remote remove origin
git remote add origin https://github.com/mg1988/focusguard-app.git
git push -u origin main
```

### Q2: 推送失败，需要 Token

**解决**：
1. 访问：https://github.com/settings/tokens
2. 生成新 Token（勾选 `repo` 权限）
3. 使用 Token 代替密码

### Q3: 推送成功但网站没更新

**解决**：
- 等待 2-5 分钟
- 访问：https://github.com/mg1988/focusguard-app/actions
- 查看部署状态
- 强制刷新浏览器（Cmd + Shift + R）
