# FocusGuard App Store 素材清单

## 📱 App Store 截图要求

### 必需尺寸（6.5 英寸显示屏）
需要准备以下尺寸的截图（至少 2 张，最多 10 张）：

#### 12.9 英寸 iPad Pro (第 6 代) - **必须**
- **分辨率**: 2048 x 2732 像素
- **用途**: App Store 主要展示

#### 13 英寸桌面显示器 - **必须**
- **分辨率**: 2880 x 1800 像素 或 2560 x 1600 像素
- **用途**: App Store 展示

#### 15 英寸桌面显示器 - **推荐**
- **分辨率**: 2880 x 1800 像素
- **用途**: 更大屏幕展示

### 截图内容建议（按顺序）

1. **主界面 - 专注模式**
   - 显示专注计时器和状态
   - 展示实时检测状态
   - 突出简洁的 UI 设计

2. **统计数据页面**
   - 展示 7 天历史数据
   - 显示专注效率分析
   - 图表和数据可视化

3. **抓拍相册功能**
   - 展示走神/瞌睡抓拍的照片
   - 隐私打码效果展示
   - 照片管理界面

4. **坐姿检测功能**
   - 实时坐姿状态显示
   - 姿势历史记录
   - 渐进式提醒界面

5. **设置页面**
   - 灵敏度调节
   - 通知设置
   - 多语言切换
   - 隐私承诺展示

### 截图制作工具推荐

- **Xcode**: 内置截图工具
- **CleanShot X**: 专业的截图工具
- **Mockup 工具**: 
  - appletoscreenshot.com
  - previewed.app
  - shots.pro

```

icon_32x32.png          (32x32)
icon_32x32@2x.png       (64x64)
icon_128x128.png        (128x128)
icon_128x128@2x.png     (256x256)
icon_256x256.png        (256x256)
icon_256x256@2x.png     (512x512)
icon_512x512.png        (512x512)
icon_512x512@2x.png     (1024x1024)  ← App Store Connect 需要
```

### 设计规范
- ✅ 1024x1024 像素（最大）
- ✅ RGB 颜色空间
- ✅ 不带 Alpha 通道
- ✅ 正方形，无圆角
- ✅ 不要添加"App Store"字样
