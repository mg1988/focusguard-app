# FocusGuard MVP 产品文档

# 一、产品概述

FocusGuard 是一款面向桌面平台的极简专注辅助工具，基于原生 SwiftUI 开发，通过摄像头面部识别技术，实时检测用户走神状态并发出提醒，同时统计专注时长与走神次数，帮助用户提升专注力。MVP 版本聚焦核心功能，纯本地运行、无后端依赖、不上传任何数据，主打轻量化、易上手、高隐私，适配原生交互风格，可快速上架 App Store 实现变现。

核心定位：桌面端轻量化专注辅助工具（MVP 版本，聚焦"检测-提醒-统计"核心闭环）

目标用户：学生、办公人群、需要提升专注力的桌面用户

核心价值：无需复杂操作，一键启动专注检测，实时提醒走神，直观统计专注数据，保护用户隐私。

# 二、核心功能（MVP 必选，无冗余）

## 2.1 走神检测功能

- 触发逻辑：通过电脑前置摄像头，利用 Vision 框架进行面部识别，当用户面部离开摄像头检测范围（判定为走神），达到设定时长后触发提醒。

- 灵敏度调节：支持 3 档灵敏度调节（Low/Medium/High），对应不同走神判定时长（Low：5秒、Medium：3秒、High：1秒），用户可根据自身需求切换。

- 检测控制：一键启动/停止检测，启动后开始专注计时，停止后计时暂停，检测状态实时显示。

## 2.2 提醒功能

- 提醒方式：系统原生通知提醒，包含标题（Focus Alert）和提示文案（Get back to work!），伴随默认提示音，不打扰用户正常操作。

- 触发条件：仅当检测到用户走神且达到设定时长时触发，提醒后自动重置走神计时，避免重复提醒。

## 2.3 数据统计功能

- 核心数据：今日专注时长（实时计时，格式为 00:00:00）、今日走神次数（检测到走神一次自动+1）。

- 数据存储：纯本地存储（UserDefaults），重启应用后数据不丢失，无需登录，无云端同步。

## 2.4 界面与交互

- 主界面：磨砂玻璃风格（贴合原生设计），包含产品名称、数据卡片（专注时长、走神次数）、检测开关、灵敏度调节控件。

- 交互逻辑：开关切换检测状态，灵敏度调节仅在检测暂停时可操作，数据实时刷新，视觉反馈清晰（检测中/暂停状态区分）。
